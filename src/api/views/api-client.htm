Welcome to UrbanSherpa REST API client

In the future it will not be required to enter updated_by_user_id as in order to use the API a user will need to authenticate. The user_id related to the authenticated user will be used as value for this field

Create Admin User

Adds a dbAdmin user. If a user with id=1 already exists, then user will be updated with the given email. DB Admin user should be the 1st user in the system with priviliges that systemAdmin users don't have. Like enable an account to use US system.




Add User

Adds a user with the given username and role.For example, a user with owner role or systemAdmin role

Roles allowed: dbAdmin, owner, systemAdmin, manager, sherpa.






Get User by username

Get a user with the given username (email address).




Create Account

Creates an account for the given owner. The owner is the person who pays for the license to use US System. Please make sure you have created a user for the owner before creating an account.
